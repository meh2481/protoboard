# Protoboard
Prototyping board for pi pico development, with some useful features:
- SD card reader
- Test LEDs
- DAC/Speaker
- Pushbuttons
- OLED Display
- 12V->5V buck converter
- USB-C/MicroUSB power in
- Pico pinout protoboard
